# BogeBook
#####实现追书推荐、排行榜检索、搜索书籍、分类检索、标签检索、模拟翻页效果、文章阅读、缓存章节等功能
###项目简介
一款遵循Material Design风格的小说阅读器
* Android一些次新的技术的一个合集示例
* API均来自追书神器，纯属共享学习之用，不得用于商业！！
* 使用MVP的设计模式
* RxJava，Retrofit2，Dagger2等一些主流开源框架的使用
* Glide加载图片
